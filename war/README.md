# 랜덤 타워 디펜스 (Web Version) 프로젝트 기획안

## 1. 개요
스타크래프트 유즈맵 '랜덤 타워 디펜스(랜타디)'를 웹 브라우저에서 즐길 수 있도록 이식하는 프로젝트입니다. HTML5 Canvas를 사용하여 부드러운 그래픽을 구현하고, 모바일과 PC 환경 모두를 고려한 반응형 UI로 제작합니다.

## 2. 게임 메카닉 (Game Mechanics)

### 맵 구성 (Map Layout)
- **타워 건설 구역**: 중앙에 위치한 6x6 ~ 8x8 격자(Grid).
- **적 이동 경로**: 건설 구역의 **외곽을 크게 한 바퀴 도는** 직사각형 순환(Loop) 형태.
    - 시작점(Start)에서 생성되어 맵 테두리를 따라 이동 후 시작점 근처의 종착점(End)으로 이동.

### 유닛 시스템 (Unit System)
원작과 동일한 40종 구성을 목표로 합니다.
- **등급 (5단계)**:
    1.  **노멀 (Normal)**: 8종 (White)
    2.  **매직 (Magic)**: 8종 (Blue)
    3.  **레어 (Rare)**: 8종 (Yellow)
    4.  **유니크 (Unique)**: 8종 (Purple)
    5.  **에픽 (Epic)**: 8종 (Red)
- **조합 (Merge)**: 같은 종류의 유닛 2개를 합치면 다음 등급의 **랜덤** 유닛 1개 획득.

### 웨이브 시스템 (Wave System)
- 일정 시간마다 적 유닛들이 등장.
- 적 유닛이 경로의 끝(Life Zone)에 도달하면 라이프 감소.
- 라이프 0이 되면 게임 오버.
- 보스 스테이지 존재.

## 3. UI/UX 디자인 방향
- **컨셉**: 사이버펑크/네온 (Cyberpunk/Neon). 어두운 배경에 형광색 라인으로 타워와 경로를 표현하여 세련된 느낌을 줍니다.
- **레이아웃**:
    - **중앙**: 게임 화면 (Canvas).
    - **하단/측면**: 컨트롤 패널 (유닛 뽑기 버튼, 업그레이드, 미션 정보).
    - **상단**: 정보창 (현재 라운드, 남은 적 수, 보유 골드, 라이프).

## 4. 개발 기술 스택
- **Core**: HTML5, CSS3, Vanilla JavaScript (ES6+).
- **Rendering**: HTML5 Canvas API (수백 마리의 유닛 렌더링 최적화).
- **Deploy**: GitHub Pages.

## 5. 초기 개발 로드맵
1.  기본 프로젝트 세팅 (index.html, style.css).
2.  맵 및 유닛 렌더링 엔진 구현.
3.  유닛 소환 및 드래그 앤 드롭 조합 기능 구현.
4.  적 웨이브 및 길찾기 알고리즘 구현.
5.  UI 스타일링 및 연동.
